Sirius Developer Manual

This document gives an overview of the internals of the Sirius platform, and describes the main APIs and extension points.

  • Architecture Overview provides a high-level view of how Sirius is structured.
  • Architecture and Concepts describes the general architecture of the system, the main concepts and top-level entry points in the APIs.
  • Meta-models gives a high-level description of the various meta-models used internally by Sirius, to represent both VSMs and representation files.
